# 📊 Carpeta: /backtest/

## 📌 Descripción:

Esta carpeta contiene los **resultados, bitácoras y reportes** de backtests ejecutados por el sistema Daredevil TradingBot.

---

## 🗂️ Contenido típico:

- **Resultados históricos de backtests**
- **Parámetros utilizados en cada prueba**
- **Observaciones y notas personales**
- **Logs de ejecución**

---

## ✅ Reglas de uso:

- Cada backtest importante debe tener su propio archivo `.md` o `.txt` donde se detallen:
  - Fechas de ejecución
  - Estrategia utilizada
  - Resultados clave
  - Observaciones

---

## 🦊 Ejemplo de archivos esperados:

- `Whale_Backtest_v1.md`
- `BTCUSDT_Backtest_MultiFrame.md`
- `Resultados_Estrategia_X.txt`
