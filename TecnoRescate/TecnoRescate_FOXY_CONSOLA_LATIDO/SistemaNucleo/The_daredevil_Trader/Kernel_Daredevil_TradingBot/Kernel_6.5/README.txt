 
# 🧙‍♂️ Kernel 6.5 - Inicio del Sistema Modular Daredevil TradingBot

## 🎯 Objetivo de esta versión:

Esta versión marca el **inicio formal del desarrollo modular del Kernel**, donde se definen las primeras estructuras de carpetas, bitácoras y módulos clave para la evolución futura del proyecto.

---

## 🛠️ Contenido de Kernel 6.5:

- **BITACORA.md:**  
  Registro de avances, decisiones y cambios realizados durante esta versión.

- **KERNEL.md:**  
  Documentación técnica central que explica el funcionamiento general del sistema, las reglas de codificación modular y las dependencias.

- **módulos/:**  
  Carpeta que contendrá los primeros módulos del sistema.  
  Los módulos actuales en desarrollo son:

  - **DareSignal_X/** → Indicadores, lógica de señales y experimentos iniciales.
  - **HardFoxy/** → Scripts utilitarios y experimentos de automatización.
  - **F-Optima_Calculator/** → Primeros cálculos y experimentos de optimización de parámetros.

---

## 📌 Instrucciones para el futuro:

- Cada avance o cambio importante debe reflejarse en la **BITACORA.md**.
- Cada módulo debe tener su propia bitácora interna dentro de su carpeta.
- Evitar duplicados: todos los scripts deben estar bien organizados por carpeta según su función.

---

## ✅ Estado actual:

Este README corresponde a la estructura base de **Kernel 6.5**, que irá evolucionando en las próximas versiones (6.6, 6.7, 6.8...).
