# 🎯 Módulo F-Optima_Calculator - Optimización de Parámetros de Trading

## 📌 Descripción:

**F-Optima_Calculator** es el módulo destinado al cálculo y optimización de parámetros clave de gestión de riesgo y capital, como el tamaño óptimo de posición basado en la metodología **F-Óptima** y otros modelos de gestión monetaria.

---

## 🛠️ Funcionalidades Previstas:

- Cálculo de **F-Óptima** a partir de resultados de backtests.
- Análisis estadístico de secuencias de trades.
- Generación de reportes de optimización de riesgo.
- Integración futura con el motor de automatización **HardFoxy**.

---

## 📂 Estructura Interna del Módulo:

- `f_optima_calculator.py` → Script principal para cálculo.
- `sample_trades.csv.txt` → Ejemplo de dataset de resultados de operaciones.
- **[Por agregar:]**
  - `risk_analysis_utils.py` → Funciones auxiliares de análisis estadístico.
  - `test_foptima.py` → Script de pruebas unitarias.
