Estructurar el sistema de archivos con hipervínculos entre hojas Excel, 
como base operativa del sistema modular TecnoRescate.

📂 ARCHIVOS INVOLUCRADOS
- Formulario Cliente → Captura inicial de datos (Google Form)
- TecnoRescate_maestro → Excel con:
    · Hoja: Inventario
    · Hoja: Historial Reparación
    · Hoja: Reciclaje
- Consola de Navegación → Menú retro para flujo guiado con Foxy 🦊

🔄 FLUJO ESTILO VIEJA ESCUELA
1. Cliente completa formulario (manual o automático)
2. Datos ingresan en hoja "Inventario"
3. Cada fila de Inventario contiene:
   - N° de Orden
   - Tipo de dispositivo
   - Estado
   - Fecha de ingreso
   - 🔗 HYPERLINK a:
       • Historial Reparación (si fue intervenido)
       • Reciclaje (si no se pudo reparar)
4. Hoja "Historial Reparación":
   - Tareas realizadas, fecha, costo
5. Hoja "Reciclaje":
   - Componentes reutilizables, fecha de reciclaje

🎨 VISUAL

📁 ARCHIVO ACTUAL
